# Radium Primitives (RDF-PR)

This directory contains the technical primitives of the Radium corpus.

Primitives define the concrete mechanisms used to enforce execution governance within automated systems.

They translate the conceptual and architectural layers of Radium into enforceable technical constructs.

Examples include:

• mandate tokens
• runtime validation mechanisms
• execution receipts
• constraint verification
