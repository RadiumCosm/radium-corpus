# Radium Doctrines (RDF-CA)

This directory contains the doctrinal foundations of the Radium corpus.

Doctrines define the conceptual principles underlying execution governance in complex automated systems.

They describe the structural problems Radium addresses and establish the conceptual vocabulary used throughout the corpus.

## Documents

RDF-CA documents introduce the foundational ideas required to understand Radium’s architectural and technical components.
