# Radium Frameworks (RDF-FW)

This directory contains the architectural frameworks of the Radium corpus.

Frameworks describe how the conceptual principles defined in the doctrines translate into system architecture.

They provide structural models for implementing execution governance across complex automated infrastructures.

## Documents

RDF-FW documents describe architectural structures and interaction patterns between authority, runtime validation, and execution.
